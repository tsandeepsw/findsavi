# 🎲 findsavi

**FindSavi** is a fun and challenging **mathematical number guessing game** built with **Next.js** and hosted using static files on GitHub Pages.  

The twist? You’re not just guessing the hidden number `n` (between 1–99). Instead, you’ll be guided by clever hints of mathematical operations performed between your guess and the secret number.  

The name **Savi** comes from the idea of "saving" or "finding" the clue that helps you progress. Since you rarely guess `n` correctly on the first try, the real challenge is to guess the **resultant (Savi)** values that lead you closer to the hidden number.

---

## 🧩 How It Works

1. You try to guess a number `n` (1–99).  
2. If your guess is wrong, the computer applies one of three operations:  
   - **Addition (+)**  
   - **Subtraction (−)**  
   - **Division (÷)**  
   …between your guess and the secret number.  
3. The result of this operation is called the **resultant (Savi)**.  
4. You must now guess the **resultant (Savi)** value correctly in the next round.  
5. The game continues until you crack the logic and find the hidden number.

Think of it like solving for **x** in algebra — except here, you’re solving for the **resultant (Savi)**. That’s why it’s called **FindSavi**!

## 🤔 Difficulty vs Simplicity

When you first try playing **FindSavi**, it feels tricky and even a bit confusing. But once you crack the underlying logic, you’ll realize how straightforward it actually is. Trust me — the game is built on a **really simple concept**, and that simplicity is what makes it both fun and interesting!
